# 入门指南

当你拿到SSOM-3588核心板和底板，我们建议你先阅读我们的入门指南，这样可以帮助你快速进入到开发阶段。

在准备开始使用SSOM-3588之前，你可能需要准备以下配件：

- 12V/2A电源适配器
- 网线
- Type-C数据线
- 支持HDMI的显示器
- HDMI线（A型接口）
- 键盘和鼠标
- MicroSD卡及读卡器（如果使用SD卡启动）

如果不需要界面显示，那你可能只需要电源适配器和Type-C数据线即可，你可以通过网线或者Type-C数据线将SSOM-3588连接到你的电脑，并通过终端工具对设备进行操作

### 1.1 Micro SD卡

SSOM-K1支持SD卡启动，并在底板预留了一个MicroSD卡插槽，如果你想使用卡启动，按照下面的烧录系统镜像步骤后，你就可以将卡直接插入到SSOM-3588底板的Micro SD卡插槽即可 
我们推荐Micro SD卡至少是32GB以上。

### 1.2 显示、鼠标和键盘
显示、鼠标和键盘
如果你想使用SSOM-3588的桌面系统，你需要为它配置一个HDMI的显示屏以及键盘和鼠标。你可以使用常规的电脑显示器以及键盘和鼠标即可。你需要确认你的HDMI线连接底板端必须为A型接口
你可能需要确认你的HDMI线连接主板端的必须要A型的接口，如果你不了解HDMI接口类型，可以参考下面信息：
