# 2. NDK Compilation of Third-party Libraries

## 2.1. Prerequisites

Download the specified version of the NDK and extract it. The NDK download path is as follows: [NDK Download Path](https://archive.spacemit.com/tools/openharmony/sdk/). The downloaded package is the complete SDK, and the NDK is included within the SDK. The extracted NDK is as follows:

```
fuqiang@snode2:/data/home/fuqiang/workspace/native$ tree -L 1

|-- build
|-- build-tools
|-- compatible_config.json
|-- llvm
|-- nativeapi_syscap_config.json
|-- ndk_system_capability.json
|-- oh-uni-package.json
`-- sysroot

4 directories, 4 files
```
